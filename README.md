# ⚽ Państwa Miasta – Piłkarska edycja

Multiplayer piłkarska gra słowna. Graj z przyjaciółmi na różnych urządzeniach w czasie rzeczywistym.

## Jak grać

1. Wpisz swój nick
2. Utwórz pokój lub dołącz kodem
3. Host ustawia liczbę kategorii, czas i rundy
4. Losowana jest litera — wpisz odpowiedzi do każdej kategorii
5. Głosujcie na odpowiedzi innych
6. Wygrywa gracz z największą liczbą punktów

## Technologie

- Firebase Realtime Database (multiplayer)
- PWA (działa jak aplikacja mobilna)
- Vanilla JS, zero zależności

## Uruchomienie lokalnie

Otwórz `index.html` w przeglądarce lub użyj lokalnego serwera:

```bash
npx serve .
```
